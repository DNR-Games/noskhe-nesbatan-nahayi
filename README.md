# قلب سالم — تولید اولیه (با مربی هوشمند)

بازی ساده‌ی وب برای عادت‌سازی سالم در پیشگیری از فشار خون بالا، همراه با یک مربی هوشمند واقعی که به سوالات پاسخ می‌دهد.

## چطور اجرا کنم؟

1. این پوشه (یا فقط فایل `index.html`) را در یک مخزن روی GitHub آپلود کن
2. از تنظیمات مخزن، بخش **Pages** را فعال کن (شاخه `main`، پوشه `/root`)
3. بعد از چند دقیقه، بازی روی آدرسی مثل `https://username.github.io/repo-name` در دسترس است

می‌توانی فایل `index.html` را مستقیم هم در مرورگر باز کنی — کاملاً کار می‌کند، چون هیچ فایل خارجی دیگری نیاز ندارد.

## بخش‌های بازی

- **ثبت عادت روزانه:** قدم، وعده‌ی کم‌نمک، ساعت خواب، فشار خون اختیاری
- **شهر نمادین:** با امتیاز جمع‌شده رشد می‌کند (ذخیره در حافظه‌ی مرورگر خودت)
- **روز پیاپی (streak):** انگیزه‌ی بازگشت روزانه
- **مربی هوشمند:** یک چت واقعی با هوش مصنوعی درباره‌ی سبک زندگی سالم

## درباره‌ی مربی هوشمند — نکته‌ی فنی مهم

چون این بازی یک صفحه‌ی ساده‌ی وب است (بدون سرور پشت صحنه)، مربی هوشمند مستقیماً از داخل مرورگر تو به Anthropic API وصل می‌شود. برای همین لازم است:

1. یک کلید API شخصی از [console.anthropic.com](https://console.anthropic.com) بسازی
2. اولین بار که وارد بخش «مربی هوشمند» می‌شوی، آن را وارد کنی

کلید فقط در `localStorage` مرورگر خودت ذخیره می‌شود و مستقیماً از مرورگر به Anthropic ارسال می‌گردد — هیچ سرور واسطه‌ای درگیر نیست. هزینه‌ی هر پیام هم از حساب API خودت کسر می‌شود (هزینه‌ی این مدل معمولاً بسیار ناچیز است).

⚠️ **ملاحظه‌ی امنیتی:** چون کلید در کد سمت مرورگر قرار می‌گیرد، این روش برای *استفاده‌ی شخصی خودت* مناسب است، نه برای انتشار عمومی بین کاربران ناشناس (چون هرکسی که کلید تو رو ببینه می‌تونه ازش استفاده کنه). برای فاز بعدی که بخوای بین کاربران دیگر هم منتشر کنی، بهتر است یک بک‌اند کوچک (مثلاً یک Cloudflare Worker یا Vercel Function) بسازیم که کلید API را پنهان نگه دارد.

## فازهای بعدی پیشنهادی

- افزودن یک بک‌اند سبک برای مخفی نگه‌داشتن کلید API (برای انتشار عمومی)
- افزودن مخزن منابع پزشکی (PDF) که مربی هوشمند از آن استفاده کند
- انتقال ذخیره‌سازی از `localStorage` به یک دیتابیس واقعی برای همگام‌سازی بین دستگاه‌ها
- چالش‌های اجتماعی بین خانواده/دوستان

## ⚠️ یادآوری مهم

این بازی و مربی هوشمندش جایگزین تشخیص یا درمان پزشکی نیستند. برای هر تصمیم درمانی حتماً با پزشک مشورت کن.
